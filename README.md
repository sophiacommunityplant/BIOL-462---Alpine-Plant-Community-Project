# Shifts in alpine plant communities due to changing climates in British Columbia parks
This script analyses long-term changes in alpine plant community diversity and composition in relation to mean annual temperature (MAT) and mean annual precipitation (MAP) across three BC provincial parks: Mount Assiniboine, Big White, and Garibaldi. The data cleaning script summarizes the percent cover data and rearranges the data matrix. The data analysis folder first creates a map of the three parks. Then, alpha diversity is summarized across the parks in a four-panel figure. PCA ordination by Park and Year is done on Hellinger-transformed cover data and presented in a figure with ellipses represenitng 95% confidence intervals. Then, how community composition/diversity has changed with temperature is answered with a mixed effects model, PERMANOVA, and linear regression. Similarly, the interaction between MAT and MAP and MAP only on diversity/community composition is analyzed using a mixed effects model, PERMANOVA, and linear regression. Finally, the percent cover of focal species is analyzed using mixed effects models with MAT, MAP, and Year.
